# PROMPTS.md

This document records the prompts used during AI-assisted development of this project.

---

## Prompt 1 – Initial application design

Build a simple AI-powered travel planning web application using Cloudflare Workers AI.  
The app should allow users to enter a travel request and receive a generated itinerary.  
Use a Worker backend and Durable Objects for conversation state.

---

## Prompt 2 – Travel planning prompt

Create a prompt for a travel planning assistant that generates a detailed day-by-day itinerary.  
The response should contain Day 1, Day 2, and Day 3 sections with specific attractions and activities.

---

## Prompt 3 – Timeline interface

Convert the AI response into a timeline layout in JavaScript.  
Each day should appear as a timeline item with a day label and travel plan content.

---

## Prompt 4 – Bullet point formatting

Transform bullet point text beginning with "-" into formatted HTML lists so the travel plan is easier to read in the timeline interface.

---

## Prompt 5 – User interface styling

Design a simple chat interface where the user message appears as a bubble and the AI response is displayed in a structured timeline format.

---
