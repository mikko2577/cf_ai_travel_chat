# cf_ai_travel_chat

AI-powered travel planning chat app built with Cloudflare Workers AI.

Live Demo: https://cf_ai_travel_chat.jiachanli910.workers.dev/

## What it does

Users enter a travel request, and the app generates a day-by-day itinerary in a timeline view.

Example:
Create a travel plan for Boston with museums, food, and scenic views.

## Tech Stack

- Cloudflare Workers
- Workers AI
- Durable Objects
- HTML, CSS, JavaScript

npm install
npx wrangler login
