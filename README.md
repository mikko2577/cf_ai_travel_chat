# cf_ai_travel_chat

AI Travel Chat is a simple AI-powered travel planning application built with **Cloudflare Workers AI**.  
The app allows users to enter a travel request and receive a **detailed day-by-day travel itinerary** generated by an AI model.

The generated itinerary is displayed in a **timeline interface** with structured daily plans and bullet-point activities.

---

## Overview

This application demonstrates how to build an **AI-powered web application** using the Cloudflare developer platform.

Users can enter travel requests such as:

Create a detailed 3-day travel plan for Boston with museums and seafood.

The AI then generates a structured itinerary including suggested attractions, activities, and food recommendations.

Example output:

Day 1
- Morning: Walk the Freedom Trail and explore historic landmarks such as Boston Common and the Massachusetts State House.
- Afternoon: Visit Quincy Market and Faneuil Hall to enjoy local food stalls and street performances.
- Evening: Have dinner at a seafood restaurant and take a walk along the waterfront.

Day 2
- Morning: Visit the Museum of Fine Arts and explore its extensive art collections.
- Afternoon: Explore the Isabella Stewart Gardner Museum and its famous courtyard.
- Evening: Walk around Back Bay and visit Newbury Street.

Day 3
- Morning: Explore Harvard Square and walk through the Harvard University campus.
- Afternoon: Walk along the Charles River or visit the MIT area.
- Evening: Finish the trip with dinner and waterfront views in the Seaport District.

## Features

- AI-powered travel itinerary generation
- Chat-style user interface
- Timeline visualization of daily travel plans
- Bullet-point formatting for clear readability
- Persistent conversation state using Durable Objects
- Serverless architecture with Cloudflare Workers

## Running Locally

Install dependencies

npm install

Login to Cloudflare

npx wrangler login

Start the development server

npx wrangler dev

Open the application in your browser

http://localhost:8787

---

## Deployment

Deploy the application to Cloudflare Workers

npx wrangler deploy

The application will be available at

https://your-worker-name.workers.dev

---

## Purpose

This project demonstrates how AI can transform natural language travel requests into structured itineraries.  
It also shows how developers can use **Cloudflare Workers AI and Durable Objects** to build lightweight AI-powered applications without managing infrastructure.
